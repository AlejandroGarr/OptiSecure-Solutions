1.1. ¿QUÉ ES OPTISECURE SOLUTIONS?

OptiSecure Solutions es una aplicación de gestión de seguridad doméstica y
empresarial desarrollada sobre Salesforce. El nombre viene de combinar "Opti"
(optimizar) con "Secure" (seguridad).

La idea de negocio es la siguiente: una empresa de seguridad necesita gestionar
las cámaras de sus clientes, las solicitudes de contratación de nuevas cámaras,
las incidencias que surjan, y al mismo tiempo ofrecer a sus
clientes un portal web donde puedan ver sus cámaras, pedir nuevas, abrir
tickets de soporte y hablar con el equipo técnico. Todo desde un mismo sitio.

Este TFG demuestra que con Salesforce se puede montar todo eso sin necesidad
de programar un backend desde cero, sin montar servidores, sin preocuparse
por la base de datos, y con un portal de clientes incluido.


1.2. ¿POR QUÉ SALESFORCE Y NO OTRA HERRAMIENTA?

Cuando se plantea un proyecto así, las alternativas típicas son:
- Hacerlo todo a mano: backend con Node.js/Django/Spring, base de datos
  PostgreSQL/MySQL, frontend con React/Angular, hosting en AWS/Azure,
  sistema de login propio, etc.
- Usar un CMS: WordPress con plugins, que se queda corto en cuanto
  necesitas lógica de negocio real.
- Usar otra plataforma low-code: como OutSystems, Mendix o Appian.

El problema de hacerlo todo a mano es que tardas meses solo en montar la
infraestructura antes de escribir una sola línea de lógica de negocio.
Necesitas configurar servidores, bases de datos, autenticación, permisos,
API REST, despliegue... Y luego mantener todo eso.

Salesforce resuelve todo esto de base:

  • Base de datos incluida: Los objetos personalizados (custom objects) son
    tablas con campos, relaciones, validaciones y fórmulas. No hay que
    instalar ni configurar nada. Crear un objeto con 7 campos, una regla
    de validación y un rollup summary tarda minutos, no horas.

  • Backend incluido: Apex es el lenguaje server-side de Salesforce (parecido
    a Java). Puedes escribir controladores, hacer queries SOQL, llamar a
    APIs externas y manejar transacciones. No necesitas un servidor aparte.

  • Portal de clientes incluido: Experience Cloud te da un portal web
    listo para que los clientes se registren, inicien sesión y accedan a
    sus datos. Con sus propios perfiles, permisos y licencias. Montar algo
    así desde cero con React + Auth0 + un backend... lleva semanas.

  • Seguridad integrada: Perfiles, permission sets, sharing rules, field-level
    security, with sharing/without sharing en Apex. No tienes que programar
    un sistema de permisos desde cero.

  • Despliegue y versionado: Con Salesforce CLI (sf) puedes hacer pull/push
    de todo el código y metadatos. El proyecto se versiona con Git como
    cualquier otro proyecto de software.

  • Integraciones: Salesforce puede llamar a APIs externas desde Apex
    (como Google Gemini en este proyecto) y también expone su propia API
    REST/SOAP para que otros sistemas se conecten.

Comparado con las alternativas low-code (OutSystems, Mendix):
  - Salesforce ya tiene el CRM debajo. No estás montando una app en el
    vacío: tienes Account, Contact, Case, Opportunity... objetos estándar
    que cubren el 80% de cualquier negocio B2B o B2C.
  - El ecosistema es enorme: AppExchange, Trailhead, comunidad de developers.
  - Es la plataforma CRM más usada del mundo, así que lo que aprendes aquí
    tiene salida laboral directa.

En resumen: Salesforce permite hacer en semanas lo que con desarrollo
tradicional llevaría meses, y encima el resultado incluye portal de clientes,
seguridad y API REST de serie.


1.3. MODELO DE NEGOCIO

OptiSecure Solutions se plantea como un producto SaaS (Software as a Service)
para empresas de seguridad que necesitan:

  • Gestionar el inventario de cámaras de seguridad de sus clientes.
  • Ofrecer a cada cliente un portal web donde ver el plano de su vivienda
    con las cámaras colocadas, solicitar nuevas cámaras y gestionar
    incidencias.
  • Tener un flujo de aprobación para las solicitudes de contratación de
    nuevas cámaras (el cliente pide → el admin aprueba → el cliente coloca
    las cámaras en el mapa).
  • Dar soporte técnico a los clientes con un sistema de tickets y chat.
  • Usar inteligencia artificial para que los clientes puedan dar órdenes
    por voz (abrir un ticket, consultar estado de cámaras, etc.).

El modelo al montarlo sobre Salesforce tiene una ventaja: es multi-tenant
de serie. Cada cliente tiene sus propios datos aislados, su propio contacto
y su propia cuenta. No hay que programar la separación de datos porque
Salesforce la hace con perfiles y sharing rules.

Además, si mañana esta empresa de seguridad quiere añadir funcionalidades
nuevas (por ejemplo conectar con sensores IoT, integrar con una pasarela de
pago, o montar una app móvil), Salesforce lo soporta sin tener que rehacer
la arquitectura.


1.4. FUNCIONALIDADES DEL PROYECTO

Lo que se ha construido en este TFG:

- Mapa interactivo del hogar: Los clientes ven el plano de su casa con sus
  cámaras colocadas. Los administradores pueden arrastrar y soltar nuevas
  cámaras sobre el plano. Las cámaras se pueden mover de sitio.

- Asistente de voz IA: Usa la Web Speech API del navegador para transcribir
  lo que dice el usuario en español. El texto se manda a Google Gemini
  (modelo gemini-2.0-flash) que entiende qué quiere hacer el usuario y
  ejecuta acciones en Salesforce (crear un caso, consultar cámaras, etc.).

- Contratación de cámaras: Los clientes rellenan un formulario para pedir
  nuevas cámaras. El admin lo revisa y aprueba o rechaza. Si se aprueba,
  el cliente puede colocar las nuevas cámaras en el mapa.

- Sistema de casos y chat: Los clientes pueden abrir tickets de soporte,
  reportar incidencias en cámaras y hablar con el equipo de soporte
  mediante un chat basado en CaseComments de Salesforce.

- Portal de clientes: Sitio web con registro, login, perfil de usuario,
  y acceso a todas las funcionalidades anteriores.

- Tests unitarios: 4 clases de test Apex que cubren los controladores
  principales, incluyendo mocks de llamadas HTTP a Google Gemini.


1.5. OBJETIVOS DEL TFG

1. Montar una aplicación completa de negocio sobre Salesforce, desde el
   modelo de datos hasta el portal de clientes.
2. Demostrar que Salesforce sirve para más que un CRM: se puede construir
   una aplicación de dominio específico (seguridad doméstica) usando la
   plataforma como base.
3. Integrar IA externa (Google Gemini) con Salesforce para procesar
   comandos de voz.
4. Crear una interfaz de usuario moderna con Lightning Web Components.
5. Cubrir el código con tests unitarios y seguir las prácticas de
   desarrollo de Salesforce (CLI, Git, SFDX).

TECNOLOGÍAS UTILIZADAS:

- Salesforce Platform: CRM y backend de datos.
- Apex: Lenguaje de programación server-side (Java-like).
- Lightning Web Components (LWC): Framework de UI basado en Web Standards.
- SOQL: Salesforce Object Query Language para consultas de datos.
- Experience Cloud: Portal de cliente con plantilla LWR.
- Salesforce CLI (sf): Herramienta de línea de comandos para despliegue.
- Google Gemini API: Modelo de IA para procesamiento de lenguaje natural.
- Web Speech API: API del navegador para reconocimiento de voz.
- HTML5/CSS3/JavaScript ES6+: Tecnologías web estándar en LWC.
- Git: Control de versiones del código fuente.

PATRONES DE DISEÑO APLICADOS:

- MVC (Model-View-Controller): Separación entre objetos (Model),
  LWC templates (View) y Apex controllers (Controller).
- Wrapper Pattern: Clases internas en Apex (CaseInfo, ChatMessage,
  UserProfileData, SolicitudAprobada, CameraData) para encapsular datos.
- Helper Pattern: CameraInsertHelper como clase auxiliar without sharing
  para operaciones DML que requieren elevación de privilegios.
- Observer Pattern: CustomEvents en LWC para comunicación entre componentes
  (ej: voicecommand event desde voiceAssistant).
- Optimistic UI: Actualización inmediata de la UI antes de confirmar
  la respuesta del servidor (ej: reposicionamiento de cámaras en el mapa).

                    4. MODELO DE DATOS — OBJETOS PERSONALIZADOS

================================================================================

El modelo de datos de OptiSecure Solutions se compone de 2 objetos
personalizados centrados en la gestión de cámaras de seguridad:

   - Camara__c: Cámaras de seguridad físicas.
   - Solicitud_Camara__c: Solicitudes de contratación de nuevas cámaras.

Además, el proyecto utiliza objetos estándar de Salesforce:
   - Account: Para las cuentas de Experience Cloud (portal users).
   - Contact: Para los contactos/usuarios del portal.
   - Case: Para los casos de soporte técnico.
   - CaseComment: Para los comentarios/chat de cada caso.
   - User: Para la gestión de usuarios y autenticación.
   - Task: Para tareas de seguimiento.

--------------------------------------------------------------------------------

de la vivienda del cliente. Cada cámara tiene coordenadas de posición para
ser mostrada sobre el mapa interactivo y una URL de vídeo para streaming.

- Label: Cámara
- Plural Label: Cámaras
- Deployment Status: Deployed
- Sharing Model: ReadWrite
- External Sharing Model: Read
- Name Field: Text (editable por el usuario)
- Search: Habilitado
- Bulk API: Habilitado
- Streaming API: Habilitado

REGLAS DE VALIDACIÓN:

1. Coordenadas_Dentro_De_Limites
   - Estado: Activa
   - Condición de error:
     OR(
       Posicion_X__c < 0,
       Posicion_X__c > 100,
       Posicion_Y__c < 0,
       Posicion_Y__c > 100
     )
   - Mensaje: "Las coordenadas X e Y deben ser un porcentaje entre 0 y 100
     para situarse dentro del mapa."
   - Propósito: Asegura que las cámaras solo se puedan posicionar dentro de
     los límites del plano (0% a 100% en ambos ejes).

Notas sobre with sharing vs without sharing:
- "with sharing": Respeta las reglas de compartición del usuario actual.
  Se usa en controladores que acceden a datos sensibles donde el usuario
  solo debe ver sus propios registros.
- "without sharing": Ignora las reglas de compartición. Se usa cuando
  los usuarios del portal necesitan acceder a datos que no podrían ver
  directamente (ej: cámaras, menú de usuario). La seguridad se implementa
  en el código Apex de forma programática (server-side filtering).

/**
 * @description Controlador Apex para el componente de reporte de incidencias
 *              de cámaras de seguridad. Permite a los usuarios de Experience
 *              Cloud crear Casos (Case) asociados a una cámara que falla.
 * @author      OptiSecure Solutions
 * @date        2026-02-26
 */
public without sharing class CameraIncidentController {

    @AuraEnabled(cacheable=true)
    public static List<Map<String, String>> getCameraOptions() {
        User currentUser = [SELECT ContactId FROM User WHERE Id = :UserInfo.getUserId() LIMIT 1];
        List<Camara__c> cameras;
        if (currentUser.ContactId != null) {
            cameras = [
                SELECT Id, Name, Zone__c
                FROM   Camara__c
                WHERE  Activa__c = TRUE
                AND    Contacto__c = :currentUser.ContactId
                ORDER BY Name ASC
            ];
        } else {
            cameras = [
                SELECT Id, Name, Zone__c
                FROM   Camara__c
                WHERE  Activa__c = TRUE
                ORDER BY Name ASC
            ];
        }
        List<Map<String, String>> options = new List<Map<String, String>>();
        for (Camara__c cam : cameras) {
            Map<String, String> opt = new Map<String, String>();
            opt.put('value', cam.Id);
            opt.put('label', cam.Name + (cam.Zone__c != null ? ' — ' + cam.Zone__c : ''));
            options.add(opt);
        }
        return options;
    }

    @AuraEnabled
    public static Id createCameraIncident(Id cameraId, String subject, String description, String priority) {
        try {
            Camara__c cam = [
                SELECT Name, Zone__c
                FROM   Camara__c
                WHERE  Id = :cameraId
                LIMIT  1
            ];

            User currentUser = [
                SELECT ContactId, Contact.AccountId
                FROM   User
                WHERE  Id = :UserInfo.getUserId()
                LIMIT  1
            ];

            Case newCase = new Case(
                Subject     = subject,
                Description = 'Cámara: ' + cam.Name + '\n'
                            + 'Zona: ' + cam.Zone__c + '\n\n'
                            + description,
                Priority    = priority,
                Status      = 'New',
                Origin      = 'Web',
                Type        = 'Mechanical',
                ContactId   = currentUser.ContactId,
                AccountId   = currentUser.ContactId != null ? currentUser.Contact.AccountId : null
            );
            insert newCase;
            return newCase.Id;
        } catch (Exception e) {
            String errorMsg = 'Error al crear la incidencia: ' + e.getMessage();
            AuraHandledException auraEx = new AuraHandledException(errorMsg);
            auraEx.setMessage(errorMsg);
            throw auraEx;
        }
    }
}

public with sharing class CaseViewerController {

    public class CaseInfo {
        @AuraEnabled public String id;
        @AuraEnabled public String caseNumber;
        @AuraEnabled public String subject;
        @AuraEnabled public String status;
        @AuraEnabled public String priority;
        @AuraEnabled public String createdDate;
        @AuraEnabled public String description;
        @AuraEnabled public String contactName;
        @AuraEnabled public String contactEmail;
    }

    public class ChatMessage {
        @AuraEnabled public String id;
        @AuraEnabled public String body;
        @AuraEnabled public String createdByName;
        @AuraEnabled public String createdDate;
        @AuraEnabled public Boolean isCurrentUser;
    }

    @AuraEnabled(cacheable=true)
    public static List<CaseInfo> getUserCases() {
        try {
            Id currentUserId = UserInfo.getUserId();
            User u = [
                SELECT ContactId FROM User
                WHERE  Id = :currentUserId
                WITH   SECURITY_ENFORCED LIMIT 1
            ];

            List<Case> cases;
            if (u.ContactId != null) {
                cases = [
                    SELECT Id, CaseNumber, Subject, Status, Priority,
                           CreatedDate, Description, Contact.Name, Contact.Email
                    FROM   Case
                    WHERE  ContactId = :u.ContactId
                    WITH   SECURITY_ENFORCED
                    ORDER BY CreatedDate DESC LIMIT 50
                ];
            } else {
                cases = [
                    SELECT Id, CaseNumber, Subject, Status, Priority,
                           CreatedDate, Description, Contact.Name, Contact.Email
                    FROM   Case
                    WHERE  OwnerId = :currentUserId
                    WITH   SECURITY_ENFORCED
                    ORDER BY CreatedDate DESC LIMIT 50
                ];
            }

            List<CaseInfo> result = new List<CaseInfo>();
            for (Case c : cases) {
                CaseInfo ci       = new CaseInfo();
                ci.id             = c.Id;
                ci.caseNumber     = c.CaseNumber;
                ci.subject        = c.Subject;
                ci.status         = c.Status;
                ci.priority       = c.Priority;
                ci.createdDate    = c.CreatedDate.format('dd/MM/yyyy HH:mm');
                ci.description    = c.Description;
                ci.contactName    = c.Contact != null ? c.Contact.Name : null;
                ci.contactEmail   = c.Contact != null ? c.Contact.Email : null;
                result.add(ci);
            }
            return result;
        } catch (Exception e) {
            throw new AuraHandledException(
                'Error al obtener los casos: ' + e.getMessage()
            );
        }
    }

    @AuraEnabled
    public static List<ChatMessage> getCaseComments(Id caseId) {
        try {
            Id currentUserId = UserInfo.getUserId();
            List<CaseComment> comments = [
                SELECT Id, CommentBody, CreatedBy.Name, CreatedDate
                FROM   CaseComment
                WHERE  ParentId = :caseId
                WITH   SECURITY_ENFORCED
                ORDER BY CreatedDate ASC
            ];

            List<ChatMessage> result = new List<ChatMessage>();
            for (CaseComment cc : comments) {
                ChatMessage msg     = new ChatMessage();
                msg.id              = cc.Id;
                msg.body            = cc.CommentBody;
                msg.createdByName   = cc.CreatedBy.Name;
                msg.createdDate     = cc.CreatedDate.format('dd/MM/yyyy HH:mm');
                msg.isCurrentUser   = (cc.CreatedById == currentUserId);
                result.add(msg);
            }
            return result;
        } catch (Exception e) {
            throw new AuraHandledException(
                'Error al obtener los comentarios: ' + e.getMessage()
            );
        }
    }

    @AuraEnabled
    public static ChatMessage addCaseComment(Id caseId, String body) {
        try {
            CaseComment cc = new CaseComment(
                ParentId    = caseId,
                CommentBody = body,
                IsPublished = true
            );
            insert cc;

            cc = [
                SELECT Id, CommentBody, CreatedBy.Name, CreatedDate, CreatedById
                FROM   CaseComment
                WHERE  Id = :cc.Id
                WITH   SECURITY_ENFORCED LIMIT 1
            ];

            ChatMessage msg     = new ChatMessage();
            msg.id              = cc.Id;
            msg.body            = cc.CommentBody;
            msg.createdByName   = cc.CreatedBy.Name;
            msg.createdDate     = cc.CreatedDate.format('dd/MM/yyyy HH:mm');
            msg.isCurrentUser   = true;
            return msg;
        } catch (Exception e) {
            throw new AuraHandledException(
                'Error al enviar el mensaje: ' + e.getMessage()
            );
        }
    }
}

public without sharing class HouseMapController {

    @AuraEnabled(cacheable=false)
    public static List<Camara__c> getCamaras(Id contactId) {
        try {
            User currentUser = [
                SELECT ContactId FROM User
                WHERE  Id = :UserInfo.getUserId() LIMIT 1
            ];

            Id effectiveContactId;
            if (currentUser.ContactId != null) {
                effectiveContactId = currentUser.ContactId;
            } else {
                effectiveContactId = contactId;
            }

            if (effectiveContactId != null) {
                return [
                    SELECT Id, Name, Video_URL__c, Posicion_X__c,
                           Posicion_Y__c, Zone__c, Contacto__c, Contacto__r.Name
                    FROM   Camara__c
                    WHERE  Activa__c = TRUE
                    AND    Contacto__c = :effectiveContactId
                    ORDER BY Name ASC
                ];
            }
            return [
                SELECT Id, Name, Video_URL__c, Posicion_X__c,
                       Posicion_Y__c, Zone__c
                FROM   Camara__c
                WHERE  Activa__c = TRUE
                ORDER BY Name ASC
            ];
        } catch (Exception e) {
            throw new AuraHandledException(
                'Error al obtener las cámaras: ' + e.getMessage()
            );
        }
    }

    @AuraEnabled(cacheable=true)
    public static Boolean isSystemAdmin() {
        try {
            Profile p = [
                SELECT Name FROM Profile
                WHERE  Id = :UserInfo.getProfileId() LIMIT 1
            ];
            return p.Name == 'System Administrator' || p.Name == 'Administrador del sistema';
        } catch (Exception e) {
            return false;
        }
    }

    @AuraEnabled
    public static Camara__c saveDraggedCamera(String name, Decimal posX, Decimal posY, String videoUrl) {
        Camara__c cam = new Camara__c(
            Name          = name,
            Posicion_X__c = posX,
            Posicion_Y__c = posY,
            Video_URL__c  = videoUrl,
            Activa__c     = true,
            Zone__c       = 'Salón'
        );
        insert cam;
        return cam;
    }

    @AuraEnabled
    public static void deleteCamera(Id cameraId) {
        try {
            Camara__c cam = [
                SELECT Id FROM Camara__c
                WHERE  Id = :cameraId WITH SECURITY_ENFORCED LIMIT 1
            ];
            delete cam;
        } catch (Exception e) {
            throw new AuraHandledException(
                'Error al eliminar la cámara: ' + e.getMessage()
            );
        }
    }

    @AuraEnabled
    public static void updateCameraPosition(Id cameraId, Decimal newX, Decimal newY) {
        try {
            Camara__c cam = [
                SELECT Id, Posicion_X__c, Posicion_Y__c FROM Camara__c
                WHERE  Id = :cameraId WITH SECURITY_ENFORCED LIMIT 1
            ];
            cam.Posicion_X__c = newX;
            cam.Posicion_Y__c = newY;
            update cam;
        } catch (Exception e) {
            throw new AuraHandledException(
                'Error al actualizar la posición de la cámara: ' + e.getMessage()
            );
        }
    }

                        profile.billingCity       = con.Account.BillingCity;
                        profile.billingState      = con.Account.BillingState;
                        profile.billingPostalCode = con.Account.BillingPostalCode;
                        profile.billingCountry    = con.Account.BillingCountry;
                    }
                } catch (Exception accErr) {
                    System.debug('Error al obtener datos de cuenta: ' + accErr.getMessage());
                }
            }

            try {
                List<Camara__c> cameraList;
                if (u.ContactId != null) {
                    cameraList = [
                        SELECT Id, Name, Zone__c, Activa__c
                        FROM   Camara__c
                        WHERE  Contacto__c = :u.ContactId ORDER BY Name
                    ];
                } else {
                    cameraList = [
                        SELECT Id, Name, Zone__c, Activa__c
                        FROM   Camara__c
                        WHERE  OwnerId = :currentUserId ORDER BY Name
                    ];
                }
                profile.cameras = new List<CameraData>();
                Integer cameraActive = 0;
                for (Camara__c cam : cameraList) {
                    profile.cameras.add(new CameraData(cam));
                    if (cam.Activa__c) cameraActive++;
                }
                profile.totalCameras    = cameraList.size();
                profile.activeCameras   = cameraActive;
                profile.inactiveCameras = cameraList.size() - cameraActive;
            } catch (Exception camErr) {
                System.debug('Error al obtener cámaras: ' + camErr.getMessage());
                profile.cameras = new List<CameraData>();
                profile.totalCameras = 0;
                profile.activeCameras = 0;
                profile.inactiveCameras = 0;
            }

        } catch (Exception e) {
            profile.name     = UserInfo.getName();
            profile.email    = UserInfo.getUserEmail();
            profile.initials = buildInitials(UserInfo.getName());
            profile.cameras         = new List<CameraData>();
            profile.totalCameras    = 0;
            profile.activeCameras   = 0;
            profile.inactiveCameras = 0;
        }

        profile.isAdmin = checkIsAdmin();
        return profile;
    }

    private static Boolean checkIsAdmin() {

    private static String buildInitials(String fullName) {
        if (String.isBlank(fullName)) return '??';
        List<String> parts = fullName.trim().split('\\s+');
        String result = parts[0].substring(0, 1).toUpperCase();
        if (parts.size() > 1) {
            result += parts[parts.size() - 1].substring(0, 1).toUpperCase();
        }
        return result;
    }

            cams = [SELECT Id FROM Camara__c WHERE Id = :cameraId AND Contacto__c = :currentUser.ContactId LIMIT 1];
        } else {
            cams = [SELECT Id FROM Camara__c WHERE Id = :cameraId LIMIT 1];

            AuraHandledException ex = new AuraHandledException('No tienes acceso a esta cámara.');
            ex.setMessage('No tienes acceso a esta cámara.');
            throw ex;

        try {
            cams[0].Name = newName.trim();
            update cams[0];
        } catch (Exception e) {
            AuraHandledException ex = new AuraHandledException('Error al renombrar: ' + e.getMessage());
            ex.setMessage('Error al renombrar: ' + e.getMessage());
            throw ex;
        }
    }

5.11 CLASE: VoiceCommandController

Archivo: force-app/main/default/classes/VoiceCommandController.cls
Sharing: with sharing
Propósito: Controlador para el Asistente de Voz IA. Recibe texto transcrito
del LWC, lo envía a Google Gemini para NLP (clasificación de intención +

INTEGRACIÓN CON GOOGLE GEMINI:
- Modelo: gemini-2.0-flash (gratuito hasta 15 RPM)
- API Key: Almacenada en Custom Label 'Gemini_API_Key'
- Endpoint: generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent
- Temperature: 0.2 (respuestas deterministas)
- Max Output Tokens: 500

SYSTEM PROMPT (NLP):
El prompt del sistema define 3 acciones posibles:
- crear_caso: Cuando el usuario pide abrir un caso, reportar un fallo.
- consultar_camaras: Cuando pregunta por el estado de las cámaras.
- no_reconocido: Cuando no se entiende la intención.

Parámetros extraídos para crear_caso:
- asunto: Resumen corto del problema
- descripcion: Descripción completa
- prioridad: High/Medium/Low (inferida del tono de urgencia)
- tipo: Mechanical/Electrical/Electronic/Structural/Other

MÉTODOS:

1. processVoiceCommand(String userText) — @AuraEnabled
   - Retorno: String (JSON con {exito, accion, mensaje, recordId})
   - Proceso:
     a) Valida que el texto no esté vacío.
     b) Llama a callGemini() con el texto del usuario.
     c) Limpia bloques markdown (`json...`) de la respuesta.
     d) Parsea el JSON de respuesta de Gemini.
     e) Ejecuta la acción correspondiente.

2. ejecutarCrearCaso(Map<String, Object> parametros) — private
   - Crea un Case con los parámetros extraídos por la IA.
   - Añade prefijo "[Creado por Asistente de Voz IA]" a la descripción.
   - Valida que la prioridad sea High/Medium/Low (default: Medium).

3. ejecutarConsultarCamaras() — private
   - Consulta cámaras activas (máx 50) con SECURITY_ENFORCED.
   - Devuelve un resumen textual: "Hay X cámaras activas: [lista]".

4. callGemini(String userText) — private
   - Realiza HTTP POST a la API de Google Gemini.
   - Construye el request body con system_instruction y contents.
   - Timeout: 30 segundos.
   - Extrae el texto de la respuesta Gemini (candidates[0].content.parts[0].text).
   - Manejo de errores HTTP: Lanza AuraHandledException si status != 200.

5. stripMarkdownCodeBlock(String text) — private
   - Elimina `json al inicio y ` al final (Gemini a veces los añade).

6. buildResponse(Boolean exito, String accion, String mensaje, Id recordId) — private
   - Construye la respuesta JSON estándar para el LWC.

public with sharing class VoiceCommandController {

    private static final String SYSTEM_PROMPT =
        'Eres un asistente de seguridad para OptiSecure Solutions. '
        + 'Analiza el texto del usuario y extrae la intención y parámetros. '
        + 'Responde SIEMPRE en formato JSON válido con esta estructura exacta:\n'
        + '{\n'
        + '  "accion": "crear_caso" | "consultar_camaras" | "no_reconocido",\n'
        + '  "parametros": {\n'
        + '    "asunto": "string — resumen corto del problema",\n'
        + '    "descripcion": "string — descripción completa",\n'
        + '    "prioridad": "High" | "Medium" | "Low",\n'
        + '    "tipo": "Mechanical" | "Electrical" | "Electronic" | "Structural" | "Other"\n'
        + '  }\n'
        + '}\n\n'
        + 'Reglas:\n'
        + '- Si el usuario pide abrir un caso, reportar un fallo, una incidencia o problema → accion = "crear_caso"\n'
        + '- Si el usuario pregunta por el estado de las cámaras → accion = "consultar_camaras"\n'
        + '- Si no entiendes la intención → accion = "no_reconocido"\n'
        + '- Infiere la prioridad según la urgencia: palabras como "urgente", "crítico", "no funciona" → High; '
        + '"parcial", "intermitente" → Medium; "menor", "estético" → Low. Por defecto → Medium.\n'
        + '- Responde SOLO con el JSON, sin texto adicional ni bloques de código.';

    @AuraEnabled
    public static String processVoiceCommand(String userText) {
        try {
            if (String.isBlank(userText)) {
                return buildResponse(false, 'no_reconocido', 'No se recibió ningún texto.', null);
            }

            String aiJson = callGemini(userText);
            aiJson = stripMarkdownCodeBlock(aiJson);

            Map<String, Object> aiResult = (Map<String, Object>) JSON.deserializeUntyped(aiJson);
            String accion = (String) aiResult.get('accion');
            Map<String, Object> parametros = (Map<String, Object>) aiResult.get('parametros');

            if ('crear_caso'.equals(accion)) {
                return ejecutarCrearCaso(parametros);
            } else if ('consultar_camaras'.equals(accion)) {
                return ejecutarConsultarCamaras();
            } else {
                return buildResponse(false, 'no_reconocido',
                    'No he podido entender tu solicitud. Intenta algo como: '
                    + '"Abre un caso porque la cámara del jardín no funciona".', null);
            }
        } catch (Exception e) {
            String errorMsg = 'Error al procesar el comando de voz: ' + e.getMessage();
            AuraHandledException auraEx = new AuraHandledException(errorMsg);
            auraEx.setMessage(errorMsg);
            throw auraEx;
        }
    }

    private static String ejecutarCrearCaso(Map<String, Object> parametros) {
        String asunto      = parametros != null ? (String) parametros.get('asunto') : 'Incidencia por voz';
        String descripcion = parametros != null ? (String) parametros.get('descripcion') : '';
        String prioridad   = parametros != null ? (String) parametros.get('prioridad') : 'Medium';
        String tipo        = parametros != null ? (String) parametros.get('tipo') : 'Other';

        Set<String> validPriorities = new Set<String>{ 'High', 'Medium', 'Low' };
        if (!validPriorities.contains(prioridad)) { prioridad = 'Medium'; }

        Case newCase = new Case(
            Subject     = asunto,
            Description = '[Creado por Asistente de Voz IA]\n\n' + descripcion,
            Priority    = prioridad,
            Status      = 'New',
            Origin      = 'Web',
            Type        = tipo
        );
        insert newCase;

        return buildResponse(true, 'crear_caso',
            'Se ha creado el caso "' + asunto + '" con prioridad ' + prioridad + '.', newCase.Id);
    }

    private static String ejecutarConsultarCamaras() {
        List<Camara__c> camaras = [
            SELECT Name, Zone__c FROM Camara__c
            WHERE  Activa__c = TRUE WITH SECURITY_ENFORCED
            ORDER BY Name ASC LIMIT 50
        ];
        if (camaras.isEmpty()) {
            return buildResponse(true, 'consultar_camaras',
                'No hay cámaras activas en el sistema.', null);
        }
        List<String> nombres = new List<String>();
        for (Camara__c cam : camaras) {
            nombres.add(cam.Name + ' (' + cam.Zone__c + ')');
        }
        return buildResponse(true, 'consultar_camaras',
            'Hay ' + camaras.size() + ' cámaras activas: ' + String.join(nombres, ', ') + '.', null);
    }

    private static String callGemini(String userText) {
        String apiKey = System.Label.Gemini_API_Key;

        Map<String, Object> requestBody = new Map<String, Object>{
            'system_instruction' => new Map<String, Object>{
                'parts' => new List<Map<String, String>>{
                    new Map<String, String>{ 'text' => SYSTEM_PROMPT }
                }
            },
            'contents' => new List<Map<String, Object>>{
                new Map<String, Object>{
                    'parts' => new List<Map<String, String>>{
                        new Map<String, String>{ 'text' => userText }
                    }
                }
            },
            'generationConfig' => new Map<String, Object>{
                'temperature' => 0.2,
                'maxOutputTokens' => 500
            }
        };

        HttpRequest req = new HttpRequest();
        req.setEndpoint('https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=' + apiKey);
        req.setMethod('POST');
        req.setHeader('Content-Type', 'application/json');
        req.setTimeout(30000);
        req.setBody(JSON.serialize(requestBody));

        Http http = new Http();
        HttpResponse res = http.send(req);

        if (res.getStatusCode() != 200) {
            throw new AuraHandledException(
                'Error en la API de Gemini (HTTP ' + res.getStatusCode() + '): ' + res.getBody()
            );
        }

        Map<String, Object> responseBody = (Map<String, Object>) JSON.deserializeUntyped(res.getBody());
        List<Object> candidates = (List<Object>) responseBody.get('candidates');
        Map<String, Object> firstCandidate = (Map<String, Object>) candidates[0];
        Map<String, Object> content = (Map<String, Object>) firstCandidate.get('content');
        List<Object> parts = (List<Object>) content.get('parts');
        Map<String, Object> firstPart = (Map<String, Object>) parts[0];
        String text = (String) firstPart.get('text');

        return text.trim();
    }

    private static String stripMarkdownCodeBlock(String text) {
        if (String.isBlank(text)) return text;
        String trimmed = text.trim();
        if (trimmed.startsWith('`' + '')) {
            Integer firstNewline = trimmed.indexOf('\n');
            if (firstNewline > 0) { trimmed = trimmed.substring(firstNewline + 1); }
            if (trimmed.endsWith('`' + '')) {
                trimmed = trimmed.substring(0, trimmed.length() - 3);
            }
        }
        return trimmed.trim();
    }

    private static String buildResponse(Boolean exito, String accion, String mensaje, Id recordId) {
        Map<String, Object> response = new Map<String, Object>{
            'exito'    => exito,
            'accion'   => accion,
            'mensaje'  => mensaje,
            'recordId' => recordId
        };
        return JSON.serialize(response);
    }
}

import { LightningElement, wire, track } from 'lwc';
import { ShowToastEvent } from 'lightning/platformShowToastEvent';
import getCameraOptions from '@salesforce/apex/CameraIncidentController.getCameraOptions';
import createCameraIncident from '@salesforce/apex/CameraIncidentController.createCameraIncident';

export default class CameraIncidentReport extends LightningElement {
    @track isModalOpen = false;
    @track isSubmitting = false;
    @track isSuccess = false;
    @track createdCaseId = '';
    @track errorMessage = '';
    @track selectedCameraId = '';
    @track subject = '';
    @track description = '';
    @track priority = 'Medium';
    @track cameraOptions = [];

    get priorityOptions() {
        return [
            { label: 'Alta — Cámara completamente inoperativa', value: 'High' },
            { label: 'Media — Cámara con funcionamiento parcial', value: 'Medium' },
            { label: 'Baja — Problema menor o estético', value: 'Low' }
        ];
    }

    get showForm() { return !this.isSuccess; }

    @wire(getCameraOptions)
    wiredCameraOptions({ data, error }) {
        if (data) {
            this.cameraOptions = data.map((opt) => ({
                label: opt.label, value: opt.value
            }));
        } else if (error) {
            console.error('Error al cargar cámaras:', JSON.stringify(error));
        }
    }

    handleOpenModal() {
        this.isModalOpen = true;
        this.isSuccess = false;
        this.isSubmitting = false;
        this.errorMessage = '';
        this._resetForm();
    }

    handleCloseModal() {
        this.isModalOpen = false;
        this.isSuccess = false;
        this.isSubmitting = false;
        this.errorMessage = '';
        this._resetForm();
    }

    handleBackdropClick() { this.handleCloseModal(); }
    handleModalClick(event) { event.stopPropagation(); }

    handleCameraChange(event) { this.selectedCameraId = event.detail.value; }
    handleSubjectChange(event) { this.subject = event.detail.value; }
    handlePriorityChange(event) { this.priority = event.detail.value; }
    handleDescriptionChange(event) { this.description = event.detail.value; }

    handleSubmit() {
        if (!this._validateForm()) return;
        this.isSubmitting = true;
        this.errorMessage = '';

        createCameraIncident({
            cameraId: this.selectedCameraId,
            subject: this.subject,
            description: this.description,
            priority: this.priority
        })
            .then((caseId) => {
                this.createdCaseId = caseId;
                this.isSuccess = true;
                this.isSubmitting = false;
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Incidencia Registrada',
                    message: 'Se ha creado el caso ' + caseId,
                    variant: 'success'
                }));
                this.dispatchEvent(new CustomEvent('incidentcreated', {
                    detail: { caseId }
                }));
            })
            .catch((error) => {
                this.isSubmitting = false;
                this.errorMessage = this._reduceErrors(error);
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Error',
                    message: 'No se pudo crear la incidencia: ' + this.errorMessage,
                    variant: 'error', mode: 'sticky'
                }));
            });
    }

    _validateForm() {
        const allValid = [
            ...this.template.querySelectorAll(
                'lightning-combobox, lightning-input, lightning-textarea'
            )
        ].reduce((valid, input) => {
            input.reportValidity();
            return valid && input.checkValidity();
        }, true);
        if (!allValid) {
            this.errorMessage = 'Por favor, complete todos los campos obligatorios.';
        }
        return allValid;
    }

    _resetForm() {
        this.selectedCameraId = '';
        this.subject = '';
        this.description = '';
        this.priority = 'Medium';
        this.createdCaseId = '';
    }

    _reduceErrors(error) {
        if (typeof error === 'string') return error;
        if (error?.body?.message) return error.body.message;
        if (error?.message) return error.message;
        return 'Error desconocido';
    }
}

import { LightningElement, api, track } from 'lwc';
import getCaseComments from '@salesforce/apex/CaseViewerController.getCaseComments';
import addCaseComment  from '@salesforce/apex/CaseViewerController.addCaseComment';

export default class CaseChatPanel extends LightningElement {
    @api recordId;
    @track chatMessages  = [];
    @track newMessage    = '';
    @track isSending     = false;
    @track isLoading     = true;
    @track errorMessage  = '';

    connectedCallback() { this._loadComments(); }

    get hasChatMessages() {
        return this.chatMessages && this.chatMessages.length > 0;
    }

    get isSendDisabled() {
        return !this.newMessage || !this.newMessage.trim() || this.isSending;
    }

    _loadComments() {
        this.isLoading = true;
        getCaseComments({ caseId: this.recordId })
            .then(data => {
                this.chatMessages = data.map(msg => ({
                    ...msg,
                    bubbleClass: msg.isCurrentUser
                        ? 'chat-bubble chat-bubble-mine'
                        : 'chat-bubble chat-bubble-other'
                }));
                this._scrollChatToBottom();
            })
            .catch(err => {
                console.error('Error al cargar comentarios:', JSON.stringify(err));
            })
            .finally(() => { this.isLoading = false; });
    }

    handleRefresh() { this._loadComments(); }

    handleMessageChange(event) { this.newMessage = event.target.value; }

    handleKeyPress(event) {
        if (event.key === 'Enter' && !event.shiftKey) {
            event.preventDefault();
            this.handleSendMessage();
        }
    }

    handleSendMessage() {
        if (this.isSendDisabled) return;
        this.isSending = true;
        this.errorMessage = '';
        const body = this.newMessage.trim();

        addCaseComment({ caseId: this.recordId, body })
            .then(msg => {
                this.chatMessages = [
                    ...this.chatMessages,
                    { ...msg, bubbleClass: 'chat-bubble chat-bubble-mine' }
                ];
                this.newMessage = '';
                this._scrollChatToBottom();
            })
            .catch(err => {
                this.errorMessage = err.body ? err.body.message : 'Error al enviar';
            })
            .finally(() => { this.isSending = false; });
    }

    _scrollChatToBottom() {
        setTimeout(() => {
            const c = this.template.querySelector('.chat-messages');
            if (c) c.scrollTop = c.scrollHeight;
        }, 100);
    }
}

import { LightningElement, track, wire } from 'lwc';
import { refreshApex } from '@salesforce/apex';
import getUserCases    from '@salesforce/apex/CaseViewerController.getUserCases';
import getCaseComments from '@salesforce/apex/CaseViewerController.getCaseComments';
import addCaseComment  from '@salesforce/apex/CaseViewerController.addCaseComment';

export default class CaseViewer extends LightningElement {
    @track isModalOpen   = false;
    @track selectedCase  = null;
    @track chatMessages  = [];
    @track newMessage    = '';
    @track isSending     = false;
    @track isLoadingChat = false;
    @track errorMessage  = '';
    _wiredCasesResult;
    @track cases = [];

    @wire(getUserCases)
    wiredCases(result) {
        this._wiredCasesResult = result;
        if (result.data) this.cases = result.data;
        else if (result.error) console.error('Error:', JSON.stringify(result.error));
    }

    get hasCases() { return this.cases && this.cases.length > 0; }
    get showList() { return !this.selectedCase; }
    get showDetail() { return !!this.selectedCase; }
    get hasChatMessages() { return this.chatMessages && this.chatMessages.length > 0; }
    get isSendDisabled() { return !this.newMessage || !this.newMessage.trim() || this.isSending; }
    get caseCount() { return this.cases ? this.cases.length : 0; }

    get caseList() {
        return this.cases.map(c => ({
            ...c,
            priorityClass: 'priority-badge priority-' + (c.priority || 'medium').toLowerCase(),
            statusClass: 'status-badge status-' + (c.status || 'new').toLowerCase().replace(/\s/g, '-')
        }));
    }

    get detailPriorityClass() {
        if (!this.selectedCase) return '';
        return 'priority-badge priority-' + (this.selectedCase.priority || 'medium').toLowerCase();
    }

    get detailStatusClass() {
        if (!this.selectedCase) return '';
        return 'status-badge status-' + (this.selectedCase.status || 'new').toLowerCase().replace(/\s/g, '-');
    }

    handleOpenModal() {
        this.isModalOpen = true;
        this.selectedCase = null;
        this.chatMessages = [];
        this.newMessage = '';
        refreshApex(this._wiredCasesResult);
    }

    handleCloseModal() {
        this.isModalOpen = false;
        this.selectedCase = null;
        this.chatMessages = [];
    }

    handleBackdropClick() { this.handleCloseModal(); }
    handleModalClick(event) { event.stopPropagation(); }

    handleCaseClick(event) {
        const caseId = event.currentTarget.dataset.id;
        const found = this.cases.find(c => c.id === caseId);
        if (found) {
            this.selectedCase = found;
            this._loadComments(caseId);
        }
    }

    handleBack() {
        this.selectedCase = null;
        this.chatMessages = [];
        this.newMessage = '';
    }

    _loadComments(caseId) {
        this.isLoadingChat = true;
        getCaseComments({ caseId })
            .then(data => {
                this.chatMessages = data.map(msg => ({
                    ...msg,
                    bubbleClass: msg.isCurrentUser
                        ? 'chat-bubble chat-bubble-mine'
                        : 'chat-bubble chat-bubble-other'
                }));
                this._scrollChatToBottom();
            })
            .catch(err => console.error('Error:', JSON.stringify(err)))
            .finally(() => { this.isLoadingChat = false; });
    }

    handleMessageChange(event) { this.newMessage = event.target.value; }

    handleKeyPress(event) {
        if (event.key === 'Enter' && !event.shiftKey) {
            event.preventDefault();
            this.handleSendMessage();
        }
    }

    handleSendMessage() {
        if (this.isSendDisabled || !this.selectedCase) return;
        this.isSending = true;
        this.errorMessage = '';

        addCaseComment({ caseId: this.selectedCase.id, body: this.newMessage.trim() })
            .then(msg => {
                this.chatMessages = [
                    ...this.chatMessages,
                    { ...msg, bubbleClass: 'chat-bubble chat-bubble-mine' }
                ];
                this.newMessage = '';
                this._scrollChatToBottom();
            })
            .catch(err => {
                this.errorMessage = err.body ? err.body.message : 'Error';
            })
            .finally(() => { this.isSending = false; });
    }

    _scrollChatToBottom() {

        }, 100);
    }

import { LightningElement, track } from 'lwc';
import { ShowToastEvent } from 'lightning/platformShowToastEvent';
import getSolicitudesPendientes from '@salesforce/apex/ContratacionCamaraController.getSolicitudesPendientes';
import aprobarSolicitud from '@salesforce/apex/ContratacionCamaraController.aprobarSolicitud';
import rechazarSolicitud from '@salesforce/apex/ContratacionCamaraController.rechazarSolicitud';

const BADGE_MAP = {
    'Pendiente':   'badge badge-pending',
    'En Revision': 'badge badge-review'
};

export default class ContractApproval extends LightningElement {
    @track solicitudes = [];
    @track isLoading = false;
    @track processingId = null;

    get hasSolicitudes() {
        return this.solicitudes && this.solicitudes.length > 0;

                    variant: 'error'
                }));
            })
            .finally(() => { this.processingId = null; });
    }

    handleRechazar(event) {
        const solId = event.currentTarget.dataset.id;
        this.processingId = solId;
        rechazarSolicitud({ solicitudId: solId })
            .then(() => {
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Solicitud rechazada', variant: 'warning'

                this._loadSolicitudes();
            })
            .catch((err) => {

                    title: 'Error', message: err.body?.message || err.message,
                    variant: 'error'
                }));
            })
            .finally(() => { this.processingId = null; });
    }

    _loadSolicitudes() {
        this.isLoading = true;
        getSolicitudesPendientes()
            .then((data) => {
                this.solicitudes = data.map((sol) => ({

                    badgeClass: BADGE_MAP[sol.Estado__c] || 'badge',

                        ? new Date(sol.Fecha_Solicitud__c).toLocaleDateString('es-ES')

            })
            .catch(() => { this.solicitudes = []; })
            .finally(() => { this.isLoading = false; });
    }

import { LightningElement, track } from 'lwc';
import getAllSolicitudes from '@salesforce/apex/ContratacionCamaraController.getAllSolicitudes';

const BADGE_MAP = {
    'Pendiente':   'badge badge-pending',
    'En Revision': 'badge badge-review',
    'Aprobada':    'badge badge-approved',
    'Rechazada':   'badge badge-rejected',
    'Completada':  'badge badge-completed'
};

const ALL_STATES = ['Todos', 'Pendiente', 'En Revision', 'Aprobada', 'Rechazada', 'Completada'];

export default class ContractHistory extends LightningElement {
    @track _allSolicitudes = [];
    @track solicitudes = [];
    @track isLoading = false;
    @track activeFilter = 'Todos';

    get filterOptions() {
        return ALL_STATES.map((s) => ({
            label: s, value: s,
            cssClass: 'filter-btn' + (this.activeFilter === s ? ' filter-btn--active' : '')
        }));
    }

    get hasSolicitudes() { return this.solicitudes && this.solicitudes.length > 0; }
    get totalCount() { return this._allSolicitudes.length; }
    get filteredCount() { return this.solicitudes.length; }

    connectedCallback() { this._load(); }
    handleRefresh() { this._load(); }

    handleFilter(event) {
        this.activeFilter = event.currentTarget.dataset.value;
        this._applyFilter();
    }

    _load() {
        this.isLoading = true;
        getAllSolicitudes()
            .then((data) => {
                this._allSolicitudes = data.map((sol) => ({
                    ...sol,
                    badgeClass: BADGE_MAP[sol.Estado__c] || 'badge',
                    solicitante: sol.Nombre_Solicitante__c || '—',
                    cuenta: sol.Cuenta__r ? sol.Cuenta__r.Name : '—',
                    fechaFormateada: sol.Fecha_Solicitud__c
                        ? new Date(sol.Fecha_Solicitud__c).toLocaleDateString('es-ES')
                        : '—',
                    fechaModificacion: sol.LastModifiedDate
                        ? new Date(sol.LastModifiedDate).toLocaleDateString('es-ES', {
                            day: '2-digit', month: '2-digit', year: 'numeric',
                            hour: '2-digit', minute: '2-digit'
                          })
                        : '—'
                }));
                this._applyFilter();
            })
            .catch(() => { this._allSolicitudes = []; this.solicitudes = []; })
            .finally(() => { this.isLoading = false; });
    }

    _applyFilter() {
        if (this.activeFilter === 'Todos') {
            this.solicitudes = [...this._allSolicitudes];
        } else {
            this.solicitudes = this._allSolicitudes.filter(
                (s) => s.Estado__c === this.activeFilter
            );
        }
    }
}

import { LightningElement, track } from 'lwc';
import { NavigationMixin } from 'lightning/navigation';
import { ShowToastEvent } from 'lightning/platformShowToastEvent';
import getUserProfileData from '@salesforce/apex/UserMenuController.getUserProfileData';
import deleteCamera from '@salesforce/apex/UserMenuController.deleteCamera';
import renameCamera from '@salesforce/apex/UserMenuController.renameCamera';
import createSolicitudCamara from '@salesforce/apex/UserMenuController.createSolicitudCamara';

const LOGOUT_URL = '/secur/logout.jsp?retUrl=https://www.optisecure-solutions.com';

export default class CustomUserMenu extends NavigationMixin(LightningElement) {
    @track profileData = null;
    @track profileError = null;
    @track isProfileModalOpen = false;
    @track isDeleting = false;
    @track editingCameraId = null;
    @track editingCameraName = '';
    @track isRenaming = false;
    @track isLoadingProfile = false;
    @track isContractModalOpen = false;
    @track contractRows = [
        { id: '1', label: 'Cámara 1', name: '', location: '', showRemove: false }
    ];
    _contractCounter = 1;
    _globalStyleEl = null;

    connectedCallback() { this.loadProfile(); }
    disconnectedCallback() { this._removeGlobalOverlay(); }

    _applyGlobalOverlay() {
        if (this._globalStyleEl) return;
        const style = document.createElement('style');
        style.setAttribute('data-profile-overlay', 'true');
        style.textContent = '...';  // z-index overrides para componentes hermanos
        document.head.appendChild(style);
        this._globalStyleEl = style;
    }

    _removeGlobalOverlay() {
        if (this._globalStyleEl) {
            this._globalStyleEl.remove();
            this._globalStyleEl = null;
        }
    }

    loadProfile() {
        this.isLoadingProfile = true;
        getUserProfileData()
            .then((data) => { this.profileData = data; this.profileError = null; })
            .catch((error) => { this.profileData = null; this.profileError = error; })
            .finally(() => { this.isLoadingProfile = false; });
    }

    // [Getters computados: hasAccountData, isAdmin, cameraList, slaBadgeClass,
    //  activeBadgeClass, activeLabel, priorityBadgeClass, priorityLabel,
    //  formattedAddress, cameraPercentage, cameraBarStyle]

    handleMenuSelect(event) {
        switch (event.detail.value) {
            case 'profile':
                this.isProfileModalOpen = true;
                this._applyGlobalOverlay();
                break;
            case 'contract':
                this._contractCounter = 1;
                this.contractRows = [
                    { id: '1', label: 'Cámara 1', name: '', location: '', showRemove: false }
                ];
                this.isContractModalOpen = true;
                this._applyGlobalOverlay();
                break;
            case 'logout':
                window.location.replace(LOGOUT_URL);
                break;
        }
    }

    handleDeleteCamera(event) {
        event.stopPropagation();
        const cameraId = event.currentTarget.dataset.id;
        const cameraName = event.currentTarget.dataset.name;
        if (!cameraId || this.isDeleting) return;
        if (!confirm('¿Eliminar cámara "' + cameraName + '"?')) return;
        this.isDeleting = true;
        deleteCamera({ cameraId })
            .then(() => {
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Cámara eliminada', variant: 'success'
                }));
                this.loadProfile();
            })
            .catch(error => {
                this.dispatchEvent(new ShowToastEvent({
                    title: 'Error', message: error.body?.message, variant: 'error'
                }));
            })
            .finally(() => { this.isDeleting = false; });
    }

    handleSubmitContract() {
        const hasEmpty = this.contractRows.some(r => !r.name.trim() || !r.location.trim());
        if (hasEmpty) return; // Mostrar toast de advertencia
        this.isSubmittingContract = true;
        const payload = this.contractRows.map(r => ({
            name: r.name.trim(), location: r.location.trim()
        }));
        createSolicitudCamara({ camerasJson: JSON.stringify(payload) })
            .then(() => {
                this.isContractModalOpen = false;
                this._removeGlobalOverlay();
            })
            .finally(() => { this.isSubmittingContract = false; });
    }
}

Método loadCameras():

    loadCameras() {
        this.isLoading = true;
        getCamaras({ contactId: this._contactId })
            .then((data) => {
                this.cameras = data.map((cam) => ({
                    ...cam,
                    positionStyle: 	op: +""+{cam.Posicion_Y__c}%; left: +""+{cam.Posicion_X__c}%;+""+,
                    listItemClass:
                        'cam-list-item' +
                        (this.selectedCamera && this.selectedCamera.Id === cam.Id
                            ? ' cam-list-item--selected'
                            : ''),
                    isEditing: this.editingCameraId === cam.Id
                }));
            })
            .catch((error) => {
                this.cameras = [];
                const msg = error && error.body ? error.body.message : 'Error al cargar cámaras';
                this.inlineMessage = 'Error al cargar cámaras: ' + msg;
                this.inlineMessageType = 'error';
                this.dispatchEvent(
                    new ShowToastEvent({ title: 'Error', message: msg, variant: 'error' })
                );
            })
            .finally(() => { this.isLoading = false; });
    }

Este método es el motor de carga del componente. Llama imperativamente a getCamaras
pasando el contactId del usuario actual. Cada cámara recibida se enriquece con:
  - positionStyle: string CSS inline "top: Y%; left: X%;" para la posición absoluta.
  - listItemClass: clase CSS condicional que marca la cámara seleccionada con fondo resaltado.
  - isEditing: booleano que indica si esa cámara concreta está en modo edición inline.


Wire isSystemAdmin:

    @wire(isSystemAdmin)

            this.isAdmin = true;

        } else if (error) {
            this.isAdmin = false;
        }
    }

Si el usuario es System Administrator, se activa el modo admin y se inicializan 3 chinchetas

    connectedCallback() {

El ciclo de vida inicializa:

Limpia el interval y elimina los listeners globales al destruir el componente.

Sistema de Drag & Drop:

El sistema de drag & drop es la parte más compleja del componente.
Permite a los administradores (y usuarios con solicitud aprobada) arrastrar chinchetas
desde un tray lateral y soltarlas sobre el plano de la casa para posicionar nuevas cámaras.

    handleDragStart(event) — Se dispara al comenzar a arrastrar una chincheta del tray.
        Registra el pinId y marca _isDraggingExisting = false.

    handleDroppedPinDragStart(event) — Se dispara al arrastrar una chincheta ya soltada.
        Marca _isDraggingExisting = true para reposicionarla en vez de crear nueva.

    handleExistingCameraDragStart(event) — Se dispara al arrastrar una cámara existente.
        Solo funciona si canDragDrop es true.

    handleDragOver(event) — Previene el comportamiento por defecto y activa isDragOver.

    handleDragLeave() — Desactiva isDragOver.

    handleDrop(event) — Lógica principal del drop:
        const mapRect = event.currentTarget.getBoundingClientRect();
        const posX = ((event.clientX - mapRect.left) / mapRect.width) * 100;
        const posY = ((event.clientY - mapRect.top) / mapRect.height) * 100;

    El cálculo convierte las coordenadas del mouse en porcentajes relativos al mapa.
    Según si es una chincheta existente o nueva:

    A) Chincheta existente (_isDraggingExisting = true):
       - Si es una cámara real (existe en this.cameras), actualiza posición vía Apex.
       - Si es una chincheta dropped, solo actualiza sus coordenadas localmente.

    B) Chincheta nueva (del tray):
       - La elimina del tray (draggablePins).
       - La añade a droppedPins con las coordenadas calculadas.

    handleDroppedPinClick(event) — Al hacer clic en una chincheta soltada:
        Abre el fullscreen con su video embed.

    handleSaveDroppedPins() — Persiste todas las chinchetas soltadas:

        Si hay solicitud aprobada activa:
            const camarasData = this.droppedPins.map((pin) => ({
                name: pin.name,
                posX: parseFloat(pin.posX),
                posY: parseFloat(pin.posY),
                videoUrl: pin.embedUrl
            }));

            saveCamarasContratadas({
                solicitudId: this.solicitudActiva.solicitudId,
                camarasJson: JSON.stringify(camarasData)
            })

        Se serializa el array de cámaras como JSON y se envía a Apex junto con el
        ID de la solicitud. Apex crea los registros Camara__c y marca la solicitud
        como completada.

        Si NO hay solicitud aprobada (admin directo):
            Se usa Promise.all para guardar cada chincheta individualmente con
            saveDraggedCamera({ name, posX, posY, videoUrl }).

    handleAddPin() — Añade una chincheta nueva al tray (solo admin).
        Incrementa _pinCounter y genera un vídeo aleatorio.

    handleRemoveTrayPin(event) — Elimina chincheta del tray sin guardar.

    handleRemoveDroppedPin(event) — Elimina chincheta ya soltada sobre el mapa.


Métodos privados helpers:

    _initDraggablePins() — Inicializa 3 chinchetas con vídeos aleatorios para el tray.

    _checkSolicitudAprobada() — Comprueba si el usuario tiene solicitud de contratación
        aprobada activa. Si la tiene, inicializa las chinchetas según el número contratado.

    _initSolicitudPins(numCamaras) — Genera N chinchetas con nombres "Cámara contratada X"
        y vídeos aleatorios para la solicitud aprobada.

    _updateExistingCameraPosition(camId, posX, posY) — Optimistic update de posición:
        1. Actualiza coordenadas localmente en this.cameras.
        2. Llama a updateCameraPosition Apex.
        3. Si éxito, recarga. Si error, rollback al array previo.

    _openFullscreen(cam) — Abre el visor fullscreen:
        const idx = this.cameras.findIndex((c) => c.Id === cam.Id);
        const fallback = FALLBACK_VIDEOS[idx % FALLBACK_VIDEOS.length];
        this.fullscreenVideoUrl = cam.Video_Url__c || fallback;
        Usa el Video_Url__c si existe; si no, selecciona un fallback por índice.

    _handleKeyDown(event) — Cierra menú contextual o fullscreen con tecla ESC.

    _closeContextMenu() — Cierra el menú contextual al hacer clic fuera de él.

    handleStartRename / handleRenameInputChange / handleCancelRename / handleConfirmRename:
        Sistema de renombrado inline de cámaras en la sidebar, idéntico al de customUserMenu.

    _updateDateTime() — Actualiza currentDateTime con formato español "dd/mm/yyyy · hh:mm:ss".

6.9. Componente voiceAssistant — Asistente de Voz con IA
─────────────────────────────────────────────────────────

El componente voiceAssistant implementa un asistente de voz con inteligencia artificial
que permite a los usuarios interactuar con Salesforce mediante comandos de voz naturales.
Utiliza la Web Speech API (SpeechRecognition) para transcribir audio a texto y envía el
texto transcrito al backend Apex (VoiceCommandController) que, a su vez, llama a la API
de Google Gemini para interpretar y ejecutar el comando.

Funcionalidades principales:
• Botón de micrófono como trigger flotante en el footer del mapa.
• Modal con interfaz de grabación: micrófono central con animación de pulso.
• Transcripción en tiempo real con Web Speech API (SpeechRecognition).
• Edición manual del texto transcrito antes de enviar.
• Envío a Apex/Gemini para procesamiento con IA.
• Vista de éxito con resultado de la ejecución del comando.
• Liberación segura del MediaStream del micrófono al finalizar.

Flujo de ejecución:
  Usuario pulsa micrófono → Solicita permiso MediaStream → Captura audio → 
  SpeechRecognition transcribe → Usuario revisa/edita → Envía a Apex →
  VoiceCommandController procesa con Gemini → Respuesta JSON parseoada →
  Si éxito: toast + CustomEvent 'voicecommand' → Si error: mensaje

    get micButtonClass() — Clase CSS del botón de micrófono (activo/inactivo).
    get micButtonTitle() — Tooltip: "Detener grabación" o "Asistente de voz".
    get modalMicClass()  — Clase CSS del botón grande dentro del modal.

Handlers del Micrófono:

    handleToggleListening() — Alterna entre iniciar y detener la grabación.

    async _startListening() {
        // 1. Solicita acceso al micrófono
        this._micStream = await navigator.mediaDevices.getUserMedia({ audio: true });

        // 2. Verifica soporte de SpeechRecognition
        const SpeechRecognition = window.SpeechRecognition || window.webkitSpeechRecognition;
        if (!SpeechRecognition) {
            this._releaseMic();
            this.errorMessage = 'Tu navegador no soporta reconocimiento de voz.';
            return;
        }

        // 3. Configura SpeechRecognition
        this._recognition = new SpeechRecognition();
        this._recognition.lang = 'es-ES';        // Idioma español
        this._recognition.continuous = true;       // Grabación continua
        this._recognition.interimResults = true;   // Resultados parciales

        // 4. Handler de resultados — concatena toda la transcripción
        this._recognition.onresult = (event) => {
            let finalTranscript = '';
            for (let i = 0; i < event.results.length; i++) {
                finalTranscript += event.results[i][0].transcript;
            }
            this.transcript = finalTranscript;
        };

        // 5. Handler de error — libera recursos
        this._recognition.onerror = (event) => {
            this.isListening = false;
            if (event.error !== 'aborted') {
                this.errorMessage = 'Error de reconocimiento: ' + event.error;
            }
            this._releaseMic();
        };

        // 6. Handler de fin — libera recursos
        this._recognition.onend = () => {
            this.isListening = false;
            this._releaseMic();
        };

        this._recognition.start();
        this.isListening = true;
    }

    _stopListening() — Detiene SpeechRecognition y libera el micrófono.

    _releaseMic() — Detiene todos los tracks del MediaStream:
        this._micStream.getTracks().forEach(t => t.stop());
    Esto es crítico para que el indicador de "micrófono activo" desaparezca
    del navegador y se liberen los recursos de hardware.


Handler de transcripción:

    handleTranscriptChange(event) — Actualiza el transcript al editar manualmente.


Handler de envío a IA:

    handleSendCommand() {
        if (!this.hasTranscript) {
            this.errorMessage = 'No hay texto para enviar.';
            return;
        }

        this._stopListening();
        this.isProcessing = true;

        processVoiceCommand({ userText: this.transcript })
            .then((result) => {
                this.aiResponse = JSON.parse(result);

                if (this.aiResponse.exito) {
                    this.isSuccess = true;
                    // Toast de éxito
                    // CustomEvent 'voicecommand' con accion y recordId
                    this.dispatchEvent(new CustomEvent('voicecommand', {
                        detail: { action: this.aiResponse.accion, recordId: this.aiResponse.recordId }
                    }));
                } else {
                    this.errorMessage = this.aiResponse.mensaje;
                }
            })
            .catch((error) => {
                this.errorMessage = this._reduceErrors(error);
            });
    }

    La respuesta de Apex es un String JSON que se parseo con JSON.parse.
    La estructura esperada es:
    {
        "exito": true/false,
        "mensaje": "Texto descriptivo del resultado",
        "accion": "crear_caso" (u otra acción detectada por Gemini),
        "recordId": "500xxxxxxxxxxxx" (ID del registro creado/modificado)
    }

    Si la ejecución es exitosa, se dispara un CustomEvent 'voicecommand' que el
    componente padre (houseSecurityMap) puede escuchar para realizar acciones
    adicionales como recargar datos.


Helper _reduceErrors:

    _reduceErrors(error) {
        if (typeof error === 'string') return error;
        if (error?.body?.message) return error.body.message;
        if (error?.message) return error.message;
        if (Array.isArray(error?.body)) {
            return error.body.map((e) => e.message).join(', ');
        }
        return 'Error desconocido';
    }

    Función utilitaria que extrae el mensaje de error de las diferentes
    estructuras de error que puede devolver Salesforce (AuraHandledException,
    errores de wire, errores JavaScript genéricos).

8. INTEGRACIÓN CON GOOGLE GEMINI API
========================================================================

El proyecto integra la API de Google Gemini para procesar lenguaje natural
mediante inteligencia artificial.


8.1. Configuración de la integración
─────────────────────────────────────

Remote Site Settings:
  • Nombre: Google_Gemini_API
  • URL: https://generativelanguage.googleapis.com
  • Activo: true

Custom Label:
  • Nombre: Gemini_API_Key
  • Valor: [La API key de Google Gemini — almacenada como Custom Label para
           seguridad y facilidad de rotación]

Se usa un Custom Label para guardar la API key en vez de ponerla directamente
en el código. Así se puede cambiar la clave sin tener que hacer deploy y no
queda hardcodeada en los archivos.


8.2. Flujo de la integración
────────────────────────────

1. El usuario habla o escribe un comando en voiceAssistant (LWC).
2. El transcripción se envía a VoiceCommandController.processVoiceCommand (Apex).
3. VoiceCommandController construye el prompt con instrucciones contextuales para Gemini.
4. Se envía una petición POST HTTPS a:
   https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=XXX

5. Gemini analiza el texto y devuelve una respuesta JSON con:
   {
     "accion": "crear_caso",
     "asunto": "Router no funciona",
     "descripcion": "El cliente reporta problemas con su router",
     "prioridad": "High"
   }

6. VoiceCommandController parseo la respuesta de Gemini.
7. Según la acción detectada, ejecuta operaciones DML (crear Case, etc.).
8. Devuelve al LWC un JSON con {exito, mensaje, accion, recordId}.


8.3. Modelo utilizado
────────────────────

• Modelo: gemini-2.0-flash
• Tipo: Text generation (generateContent)
• Endpoint: /v1beta/models/gemini-2.0-flash:generateContent
• Formato de petición: JSON con "contents" array
• Formato de respuesta: JSON con "candidates" array

Gemini 2.0 Flash es un modelo rápido y gratuito hasta cierto límite de peticiones,
que viene bien para procesar comandos de voz donde se necesita una respuesta rápida.

El modelo de seguridad de OptiSecure Solutions se basa en la combinación de

9.2. Verificaciones en Apex
───────────────────────────

HouseMapController.isSystemAdmin():
  @AuraEnabled(cacheable=true)
  public static Boolean isSystemAdmin() {
      return [SELECT Profile.Name FROM User WHERE Id = :UserInfo.getUserId()].Profile.Name == 'System Administrator';
  }

Esta verificación se usa en el frontend para mostrar/ocultar controles de admin

──────────────────────────────

  • Se puede cambiar la key sin hacer deploy.

navegador del cliente, así la API key no se expone en el frontend.

Remote Site Settings controla qué dominios externos puede contactar Apex,
limitando las llamadas solo a generativelanguage.googleapis.com.

13.1. Error Mixed DML en auto-registro de comunidad

Problema: Al intentar auto-registrar un usuario en la comunidad, se producía
un error MIXED_DML_OPERATION porque el flujo de auto-registro intentaba realizar
operaciones DML tanto en objetos de setup (User) como en objetos estándar
en la misma transacción.

Solución: Se desactivó el flujo de auto-registro problemático modificando
el FlowDefinition correspondiente con activeVersionNumber = 0.


13.2. Agotamiento de licencias Customer Community Plus Login
───────────────────────────────────────────────────────────

Problema: Las 5 licencias de Customer Community Plus Login estaban agotadas
(5/5 en uso), impidiendo la creación de nuevos usuarios de portal.

Solución: Se desactivaron 2 usuarios que no estaban en uso activo, liberando
licencias para quedar en 3/5.


13.3. Incompatibilidad de header Aura en template LWR
───────────────────────────────────────────────────────

Problema: La plantilla LWR (Lightning Web Runtime) del Experience Cloud no
soporta componentes Aura en el header, lo que impedía usar un header personalizado
estándar de Aura.

Solución: Se creó el componente LWC customUserMenu que funciona como un menú
de usuario completo (perfil, contratación, logout) y se insertó directamente
como componente en el header del tema LWR.


13.4. Z-index de modales vs chinchetas del mapa
───────────────────────────────────────────────

Problema: Los modales de perfil y contratación aparecían por debajo de las
chinchetas del mapa de seguridad, quedando ocultos tras los pins.

Solución: Implementación de inyección de estilos globales (Global Style Injection)
mediante la inserción dinámica de un <style> tag en document.head desde el
lifecycle connectedCallback/disconnectedCallback del componente customUserMenu.
Esto permitió forzar z-index elevados en los modales incluso cuando el Shadow DOM
de otros componentes impedía la cascada CSS normal.

    _applyGlobalOverlay() {
        if (!this._styleTag) {
            this._styleTag = document.createElement('style');
            this._styleTag.textContent = '.camera-pin, .dropped-pin, .map-badge { z-index: 1 !important; }';
            document.head.appendChild(this._styleTag);
        }
    }

    _removeGlobalOverlay() {
        if (this._styleTag) {
            this._styleTag.remove();
            this._styleTag = null;
        }
    }

14.1. Resumen del proyecto
─────────────────────────

OptiSecure Solutions es una aplicación de gestión de seguridad doméstica
montada sobre Salesforce. El proyecto demuestra que con Salesforce se
puede levantar un negocio completo — desde el modelo de datos hasta el
portal de clientes — sin necesidad de montar servidores, bases de datos
ni sistemas de autenticación por cuenta propia.

Lo que se ha construido cubre:

• Portal de clientes con registro, login y perfil de usuario
• Mapa interactivo con drag & drop de cámaras sobre el plano de la casa
• Flujo de contratación con aprobación administrativa
• Sistema de tickets y chat entre clientes y soporte
• Asistente de voz con IA (Google Gemini) para dar órdenes por voz
• Tests unitarios con mocks de APIs externas

Todo esto se ha hecho con un equipo de una persona, en el tiempo que dura
un TFG. Con desarrollo tradicional (backend propio + frontend propio +
base de datos + hosting + autenticación + portal) habría sido imposible
llegar a este nivel de funcionalidad en el mismo plazo.

Eso no significa que Salesforce sea la solución a todo: tiene sus costes
de licencia, su curva de aprendizaje, y sus limitaciones (por ejemplo
los límites de queries SOQL por transacción, o las restricciones del
Shadow DOM en LWC). Pero para un producto B2B/B2C donde necesitas CRM,
portal y control de acceso, es difícil encontrar algo
que te dé tanto hecho de entrada.


14.2. Tecnologías utilizadas
───────────────────────────

Salesforce Platform:
  • Lightning Web Components (LWC) — Framework de componentes web
  • Apex — Lenguaje de programación server-side
  • SOQL — Lenguaje de consultas de Salesforce
  • Experience Cloud (LWR) — Portal de clientes
  • Remote Site Settings — Gestión de callouts externos
  • Custom Labels — Almacenamiento seguro de configuración

APIs Web estándar:
  • Web Speech API (SpeechRecognition) — Transcripción de voz a texto
  • MediaStream API — Acceso al micrófono del navegador
  • Drag and Drop API — Arrastrar y soltar elementos

APIs externas:
  • Google Gemini API (gemini-2.0-flash) — Procesamiento de lenguaje natural

Herramientas de desarrollo:
  • Salesforce CLI (sf) — Línea de comandos para desarrollo
  • VS Code + Salesforce Extension Pack — IDE
  • Jest + @salesforce/sfdx-lwc-jest — Testing de componentes
  • ESLint + @lwc/eslint-plugin-lwc — Linting de código
  • Prettier — Formateo de código
  • Git — Control de versiones


14.3. Líneas de trabajo futuro
─────────────────────────────

Hay varias cosas que se podrían añadir en el futuro:

1. Planos por cuenta:
   Que cada Account tenga su propio plano de vivienda, en vez de usar
   el mismo para todos.

2. Streaming real de cámaras:

   Un panel con métricas de uptime de cámaras, incidencias por zona,

   las cámaras desde el móvil.

   solo funciona en español).
